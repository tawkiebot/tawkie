# Tawkie's Voice AI Project

TypeScript-based voice AI and automation projects.

## Setup

```bash
# Install dependencies
npm install

# Build TypeScript
npm run build

# Run development
npm run dev

# Run production
npm start
```

## Structure

```
tawkie/
├── src/           # TypeScript source files
├── tests/         # Test files
├── dist/          # Compiled JavaScript
├── package.json   # Dependencies
└── tsconfig.json # TypeScript config
```

## Voice AI Stack

- **STT**: faster-whisper
- **TTS**: edge-tts
- **Audio**: PyAudio
