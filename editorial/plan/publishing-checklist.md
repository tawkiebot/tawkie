# Publishing Checklist

## Pre-Writing
- [ ] Keyword research complete
- [ ] Target URL finalized
- [ ] Competitor articles reviewed
- [ ] Outline approved

## Writing
- [ ] H1 contains primary keyword
- [ ] Problem statement clear in opening
- [ ] Real example (input/output) included
- [ ] "When this fails" section added
- [ ] Tradeoff table complete
- [ ] Starter template included
- [ ] Internal links placed (2-3 per article)

## Review
- [ ] Grammar/spell check
- [ ] Style guide compliance
- [ ] CTA placement verified (product + docs)
- [ ] Links working
- [ ] Code blocks tested

## Publish
- [ ] SEO meta description (150-160 chars)
- [ ] Featured image ready
- [ ] Social share text prepared
- [ ] Internal linking confirmed
- [ ] Published on scheduled date

## Post-Publish
- [ ] Indexing verified
- [ ] Performance tracked
- [ ] Update within 90 days
